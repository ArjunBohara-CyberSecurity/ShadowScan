# 🛡️ ShadowScan

> **Next-Gen GUI Web Auditing Toolkit For Ethical Hackers & Security Researchers**

ShadowScan Is A **Modern, GUI-First Penetration Testing Toolkit** Built For Speed, Clarity, And Control.  
It Combines **Powerful Multi-Threaded Scanning** With A **Clean Desktop Interface**, Making Security Auditing Feel Like A Premium Application.

---

## 🧠 Core Capabilities

- ⚡ **150+ Modular Security Checks**
- 🎯 **Targeted & Full Scan Modes**
- 🧵 **Multi-Threaded Engine**
- 🖥️ **Beautiful Desktop GUI**
- 📊 **Real-Time Progress Tracking**
- 📁 **Structured JSON Reporting**
- 🛑 **Start / Stop / Cancel Control**

## ⚙️ Installation

```bash
Git Clone Https://Github.Com/Your-Username/ShadowScan.Git
Cd ShadowScan

Python -M Venv Venv

# Windows
Venv\Scripts\Activate

# Linux / MacOS
Source Venv/Bin/Activate

Pip Install -R Requirements.Txt
```

---

## ▶️ Launch

```bash
Python ShadowScan.Py
```

---

## 🧪 Usage

1. Enter Target URL (`Http://` Or `Https://`)
2. Select Modules Or Click **Select All**
3. Click **Run Audit**
4. Monitor Live Scan Progress
5. Export Results Via **JSON**
